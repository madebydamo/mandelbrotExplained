What is the Mandelbrot set?

z_n+1 = z_n^2 + c
This formula captures the whole generated fracture. But where do the colors come from? And how do we generate this image?
Let us explore this wonderful Beauty in this exploratory website. Buckle your seatbelts!

What is complex number squared?

A complex number (a + bi) squared equals (a + bi)^2 = (a^2 + 2abi - b^2). This doesn't seem intuitiv.

How do we get the next iteration?

Now lets return to our original formula z_n+1 = z_n^2 + c
Important to note is that z_0 = c. But what is c?
It is the starting point of the iteration and it doesn't change.
Let us visualize the iteration for an arbitrary point.
See how some points c seem to escape to infinity and some converge.

But where does the color come from?

As you saw, some points escape to infinity, others converge.
Now how about we give every point a color, by how long he needs to escape to infinity?
Thats already how we generate the beautyful fraction.
The Mandelbrot Set contains all the points that converge. They are typically drawn as black
So the colors you see in the image are points that go to infinity by different speeds
