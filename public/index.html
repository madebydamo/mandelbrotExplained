What is the Mandelbrot set?

z_n+1 = z_n^2 + c
This formula captures the whole generated fractal. But where do the colors come from? And how do we generate this image?
Let us explore this wonderful beauty in this exploratory website. Buckle your seatbelts and scroll down!

How do we get the next iteration?

Now let's return to our original formula z_n+1 = z_n^2 + c
It is important to note that z_0 = c. But what is c?
It is the starting point of the iteration and it doesn't change.
Let us visualize the iteration for an arbitrary point.
See how some points c seem to escape to infinity and some converge.

But where does the color come from?

As you saw, some points escape to infinity, others converge.
Now, let's give every point a color, depending on how long it takes to escape to infinity.
That's how we generate this beautiful fractal.
The Mandelbrot Set contains all the points that converge. They are typically drawn as black
So the colors you see in the image are points that go to infinity at different speeds
